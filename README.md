
# FloraVision Plant Website

A responsive plant e-commerce landing page built using HTML, CSS, and JavaScript.

## Features
- Responsive Navbar
- Dropdown Menu
- Image Slider / Carousel
- Plant Product Cards
- Glassmorphism UI Design
- Smooth Layout and Modern Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Preview
This project showcases a modern plant shopping website UI with interactive sliders, product sections, and responsive navigation.

## Folder Structure
- index.html
- plantstyle.css
- plantscript.js
- plantsImages/

## How to Run
1. Download or clone the repository
2. Open `index.html` in browser
